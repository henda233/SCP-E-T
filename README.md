《SCP-E-T》

这是一款类CDDA的游戏，一款关于SCP基金会世界观的生存游戏。

目前只有单人模式，玩家可扮演D级人员、科研人员、MTF、CL、设施安保人员、清理工等非SCP角色逃离设施或者进入设施。

这也是一款伪回合制的游戏（参考CDDA）。

目前只有我一人开发中，大部分的游戏内容已经开发完成，不过仍有不少的游戏内容尚未实现。
